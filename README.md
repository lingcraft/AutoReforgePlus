## ✨ 优化

在原模组的基础上优化了：

1. 重铸窗口可自由拖动，同时也可以在设置页调整
2. 窗口位置自动记忆，支持退出重进游戏
3. 窗口增加锁定按钮，调整到自己喜欢的位置后可锁定，强迫症福音
