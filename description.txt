自动重铸优化版，一键重铸装备到指定前缀
优化说明：
    1. 重铸界面增加拖动功能，重开重铸界面、重进游戏均可自动记忆位置，设置页也可以设置窗口位置。
    2. 界面完善，部分未汉化的控件和提示进行汉化。
    3. 增加锁定按钮，可锁定界面，防止不小心拖动。
    4. 增加隐藏原始哥布林重铸界面的功能，默认开启，只显示本模组的重铸界面并隐藏原始哥布林重铸界面。
    5. 修复原模组的重铸价格和原始哥布林重铸界面的价格不一样的问题。
    6. 移除重铸选项列表中的空前缀选项。
    7. 适配其他模组的前缀稀有度颜色，修复其他模组的前缀显示为白色的问题。

Auto Reforge Plus
Optimization Notes：
    1. Added drag functionality to the reforging interface. Reopening the reforging interface and restarting the game will automatically remember the current position. The window position can also be set in the settings page.
    2. Improved the interface, some previously untranslated controls and prompts have been translated.
    3. Added a lock button to lock the interface and prevent accidental dragging.
    4. Added the ability to hide the original goblin reforging interface. Enabled by default, this only displays the reforging interface for this mod and hides the original goblin reforging interface.
    5. Fixed the issue where the reforging price in the original mod was different from the price in the original goblin reforging interface.
    6. Removed the empty prefix option from the reforging options list.
    7. Adapted the prefix rarity colors for other mods and fixed the issue where prefixes in other mods were displayed as white.
